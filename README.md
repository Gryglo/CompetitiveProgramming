# Competitive Programming

A comprehensive collection of my solutions and implementations for various competitive programming platforms, competitions, and training camps. This repository serves as a personal archive of my progress and a resource for algorithmic problem-solving in C++.

## 📂 Repository Structure

The project is organized into several directories based on the source of the problems:

- **[OI/](./OI/) & [OI_XXXIII/](./OI_XXXIII/)**: Solutions for the **Polish Olympiad in Informatics** (*Olimpiada Informatyczna*).
- **[oboz/](./oboz/)**: Problems and materials from various **Competitive Programming Camps** (*Obozy*), such as:
  - *Krzyżowa* (2023, 2025)
  - *Szklarska Poręba* (2026)
- **[CF/](./CF/)**: Solutions for **Codeforces** contests and individual problems, including specific events like *GoodBye 2025*.
- **[sparingi/](./sparingi/)**: Practice contests.

## 🛠️ Workflow

- **Language**: Primarily C++.
- **Environment**: Developed using Visual Studio Code.
- **Testing Scripts**: Shell scripts (`.sh`) for automated testing.

## 📈 Goals

- Mastering advanced algorithms and data structures.
- Improving problem-solving speed and accuracy.
- Preparing for the Polish Olympiad in Informatics (OI) and other major competitions.
